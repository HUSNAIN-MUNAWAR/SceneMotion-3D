# Recruiter Summary

SceneMotion-3D is a senior-level computer vision portfolio project. It turns videos into relative camera trajectories, sparse/dense 3D point clouds, quality diagnostics, benchmark metrics, and professional reports.

## Why it is difficult

The project uses geometry-based vision: feature matching, camera motion estimation, triangulation, scale ambiguity handling, and trajectory evaluation. These are skills used in robotics, AR/VR, autonomous systems, 3D reconstruction, and SLAM.

## Skills shown

- Computer vision and 3D geometry
- Robotics perception / SLAM concepts
- Python, OpenCV, FastAPI
- Next.js and TypeScript UI
- Docker and reproducible setup
- Testing and benchmark reporting
- Honest technical documentation

## Roles it matches

Computer Vision Engineer, Robotics Perception Engineer, SLAM Engineer, 3D Vision Engineer, AR/VR CV Engineer, AI Backend Engineer for Vision Systems.
