# Release Checklist

Before pushing:

- [ ] run `make test`
- [ ] run `make demo`
- [ ] run `make benchmark-demo`
- [ ] run `make bundle-demo`
- [ ] run `make screenshots`
- [ ] run `make validate-final`
- [ ] run `cd frontend && npm install && npm run typecheck && npm run build`
- [ ] check README links
- [ ] check Mermaid diagrams render on GitHub
- [ ] check screenshots exist
- [ ] check artifact bundle exists
- [ ] check no secrets
- [ ] check ZIP size
- [ ] check license
- [ ] check limitations are visible
- [ ] check no fake claims
